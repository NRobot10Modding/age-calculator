The GPL-3.0 license associated with this project applies to website and to it's python backend. However, you may use age_standalone.py in any way you want, but you mustn't change it AT ALL and must credit me somewhere :)

The standalone age script you see in this repo will work in any IDE or just in a CMD or terminal window, whereas the main.py REQUIRES the website front in order to work. This script accounts for leap years and even the weirdness with the month of December equalling 12, which came up for some reason.

Currently, a bug exists where inputting the current month as the month you were born on declares you -1 months old, but that will soon be fixed.
